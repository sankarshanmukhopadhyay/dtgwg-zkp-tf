# Capability readiness model

Score each capability as **absent**, **documented**, **implemented**, **tested** or **independently evidenced**.

| Capability | Required production state |
|---|---|
| Authority and delegation | Tested |
| Predicate and policy semantics | Independently evidenced |
| Issuance and status | Tested |
| Holder binding and recovery | Tested |
| Verification pipeline | Independently evidenced |
| Registry resolution | Tested |
| Privacy claim | Independently evidenced |
| Incident response | Tested |
| Recovery and migration | Tested |
| Conformance evidence | Independently evidenced |

A deployment cannot compensate for a missing authority decision with cryptographic strength.
