# Profile selection guide

Select profiles by required claim, assurance horizon, disclosure boundary, operating environment and adversary model.

| Decision | Question |
|---|---|
| Predicate | What exact proposition must be verified? |
| Attestation | Which fields determine the proposition and its assurance limit? |
| Holder binding | What proves control by the presenting holder? |
| Delegation | May an agent act, for whom, and within what scope? |
| Registry | Which authority publishes recognition and status? |
| Proving | Local, remote or hybrid, and what can each service observe? |
| Offline | What state may be cached, for how long, and under whose risk acceptance? |

Profile selection shall not silently broaden identity, personhood or authorization claims.
