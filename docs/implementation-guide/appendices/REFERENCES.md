## Normative references

- [*A Cryptographic Framework for Proof of Personhood*](https://eprint.iacr.org/2026/333) — Choudhuri, Garg, Lee, Montgomery, Policharla, Sinha (IACR ePrint 2026/333). Formalises personhood credentials (PHCs), verifiable relationship credentials (VRCs), Sybil-resistance and context-dependent unlinkability, and establishes why full unlinkability and Sybil resistance cannot coexist. Adopted as the normative reference for this task force's requirements; this repository specifies a deployment profile of the framework rather than an independent one. See [`proof-of-liveness-requirements.md`](../../../proof-of-liveness-requirements.md) and the ADRs listed in [`adr/README.md`](../adr/README.md) that cite it.
