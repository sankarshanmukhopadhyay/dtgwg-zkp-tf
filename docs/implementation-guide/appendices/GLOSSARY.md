# Glossary

**Accreditation framework:** Governance arrangement defining issuer eligibility, assurance,
audit, status, and accountability.

**Context:** Governance- and protocol-defined domain in which a predicate, nullifier, or
linkability rule applies.

**Horizon:** Time interval for which a privacy or assurance claim is intended to hold.

**Nullifier:** Deterministic, scoped output used to detect repeated use without a global identifier.

**Proof profile:** Versioned contract defining statements, inputs, outputs, privacy properties,
transcript binding, lifecycle semantics, and permitted algorithms.
