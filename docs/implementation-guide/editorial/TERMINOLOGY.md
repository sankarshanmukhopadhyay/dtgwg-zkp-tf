# Controlled terminology

| Preferred term | Avoid or qualify | Rule |
|---|---|---|
| zero-knowledge proof | ZKP on first use | Expand once per standalone document |
| scoped uniqueness | uniqueness | Always name scope and epoch |
| policy-defined personhood | personhood | Do not imply civil identity |
| holder binding | ownership, authority | State the bound key or secret |
| delegated authority | agent control | State source, scope, duration and revocation |
| relying-party policy | verifier trust | Identify policy version and authority |
| registry record | trusted registry entry | Separate publication, recognition and reliance |
| decision receipt | audit log | State contents, retention and disclosure |
| privacy horizon | permanent privacy | State duration and adversary |
| degraded path | fallback | State assurance difference explicitly |
