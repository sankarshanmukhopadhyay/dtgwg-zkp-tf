# Network and trust-zone controls

Trust zones reflect authority and data exposure, not only network segments. Authenticate and authorize every service call, restrict egress from proof and key services, isolate administrative interfaces, prevent direct public access to evidence stores and monitor policy or registry control-plane changes.

Document which components can observe requests, credentials, witnesses, presentations, status lookups and decision outcomes.
