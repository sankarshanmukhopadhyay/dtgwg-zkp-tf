# Deployment evidence profile

A deployment evidence package records topology identifier, component inventory, software versions, active profiles, policies, key custody, trust zones, data flows, threat assessment, control statement, conformance report, operational readiness, recovery and rotation exercises, residual risks and accountable owners.

The canonical machine-readable form is validated against `conformance/schemas/deployment-evidence.schema.json`.
