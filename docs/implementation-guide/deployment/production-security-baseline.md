# Production security baseline

Production deployments implement least privilege, workload identity, administrative separation, encrypted service channels, protected signing keys, signed releases, dependency provenance, vulnerability management, configuration integrity, restricted administration, rate limits, privacy-minimized logging, trusted time, encrypted backups, restoration testing, tenant isolation and emergency suspension.

{: .warning }
Availability controls shall not convert registry unavailability into an affirmative status or silently extend an expired assurance horizon.
