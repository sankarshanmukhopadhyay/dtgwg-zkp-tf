# Key and secret management

Document generation, custody, activation, rotation, suspension, recovery and destruction for issuer, registry, holder, agent and service keys. Separate signing, encryption and administrative credentials. High-impact issuer and registry changes should require dual control.

Emergency rotation defines the authority to suspend keys, publication path, verifier propagation target, overlap period and evidence required to close the incident.
