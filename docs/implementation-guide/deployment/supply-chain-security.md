# Software supply-chain security

Builds should be reproducible where practical, signed, associated with source revisions and accompanied by dependency inventories. Production admits only approved artefacts. Vulnerability response defines triage authority, exposure analysis, emergency release, rollback and evidence preservation.

A cryptographic library vulnerability triggers profile impact analysis, not merely a package upgrade.
