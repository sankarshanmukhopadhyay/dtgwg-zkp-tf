# Auditor and assessor guide

Assessment follows evidence from claim to implementation. Reviewers should verify authority records, profile and policy versions, deployment topology, threat-control mapping, conformance results, runbook exercises, residual-risk approvals and evidence integrity.

Sampling should include accepted, rejected, stale, unavailable, revoked, delegated and recovery paths. An assessment report shall distinguish design evidence, implementation evidence, operational evidence and independent observation.
