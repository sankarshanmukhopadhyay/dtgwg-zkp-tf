# Conformance and plugfest

Validate predicate semantics, canonical transcript and encodings, lifecycle and status,
cross-vendor interoperability, privacy against named adversaries, constrained devices,
fallback and downgrade, and error consistency.

Minimum topology:

- two issuer implementations;
- two wallet/prover implementations;
- two verifiers;
- registry and policy fixtures;
- positive, negative, malformed, lifecycle, and privacy tests.
