# Conformance decision backlog

The machine-readable backlog is maintained in [`conformance-decision-backlog.csv`](./conformance-decision-backlog.csv). Every non-executable protocol case identifies the missing construction, fixture or normative decision.

A blocked disposition is not a passing result. It is evidence that the expected behaviour is documented while executable proof remains unavailable. Decisions must be recorded before a case is promoted to `EXECUTABLE`.
