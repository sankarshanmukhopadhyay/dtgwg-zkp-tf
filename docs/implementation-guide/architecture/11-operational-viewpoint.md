# Operational viewpoint

Operations preserve the meaning of a verified presentation over time. Monitoring should distinguish cryptographic failures, semantic mismatches, policy rejections, stale state, revocation, delegation failures and infrastructure faults.

Incident response shall identify the affected authority boundary, suspend unsafe behavior, preserve decision evidence, publish status where authorized and support correction or redress. Recovery shall not silently reset scoped uniqueness, delegation restrictions or prior revocations.

Migration plans should define overlap periods, accepted versions, downgrade protections, archive verification and evidence retention for policy, issuer, registry and algorithm changes.
